Cet attribut permet d'indiquer la source du terme de vocabulaire contrôlé contenu dans l'élément. Il est disponible dans les sous- éléments de l'élément Vedettes et accès contrôlés <controlaccess> : Collectivité <corpname>, Nom de famille <famname>, Activité <function>, Genre et caractéristiques physiques <genreform>, Nom géographique <geogname>, Nom <name>, Fonction <occupation>, Nom de personne <persname>, Mot-matière <subject> et Titre <title>, ainsi que dans Description physique <physdesc>.
