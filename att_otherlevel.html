@otherlevel Niveau de description archivistique (valeur libre)
Valeur {text}
Parents
A. <archdesc> 
C. <c>, <c01>, <c02>, <c03>, <c04>, <c05>, <c06>, <c07>, <c08>, <c09>, <c10>, <c11>, <c12> 
Cet attribut permet de spécifier le niveau de description archivistique des unités documentaires décrites dans les éléments Description archivistique <archdesc>, Composant (non numéroté) <c> et Composant (numéroté) <c01> à <c12>, lorsqu'on ne trouve pas de terme approprié dans la liste de valeurs semi-fermée de l'attribut LEVEL (ex : "collection", "fonds", "series", etc.). Il faut alors donner à l'attribut @level la valeur "otherlevel" (autre niveau) puis saisir le terme approprié dans l'attribut @otherlevel.
