Élément permettant de décrire une version de l’instrument de recherche ou d’une autre entité bibliographique.

Quand il est utilisé dans le sous-élément Mention d’édition <editionstmt> d’un élément En-tête EAD <eadheader> ou dans le sous-élément Page de titre <titlepage> de l’élément Préliminaires <frontmatter>, l’élément <edition> fait référence à la version de l’instrument de recherche. Une nouvelle édition d’un instrument de recherche présente des ajouts ou des modifications significatifs et devrait remplacer les versions en ligne précédentes.

Quand il est utilisé dans l’élément Référence bibliographique <bibref>, l’élément <edition> précise la version d’une oeuvre publiée.
