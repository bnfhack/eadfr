Cet attribut permet de proposer un intitulé pour l'affichage d'un élément lorsqu'un intitulé significatif ne peut être déduit du nom de cet élément par la feuille de style ou lorsque l’élément Intitulé <head> n'est pas utilisable. Cet attribut est disponible pour tous les éléments contenus dans l'élément Identification et description <did>.
